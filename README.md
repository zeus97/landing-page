
# Landing page for music app

A lading page created with React.


## Technologies Used

-HTML

-CSS

-Typescript

-React JS




## Features

- Live preview
- Responsive Design


## Run Locally

Clone the project

```bash
  git clone https://github.com/zeus97/landing-page.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
